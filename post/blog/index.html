使用hugo创建博客

Jul 1, 2019
1 min read

####获取hugo

https://github.com/gohugoio/hugo/releases

下载hugo window zip 解压配置环境变量

cmd命令窗口输入hugo version查看版本号

hugo version

新建战点

hugo new site sitename

将会在当前路径下创建sitename站点

获取主题

https://themes.gohugo.io/

cd sitename
git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c

进入站点，把m10c theme下载到themes下

新建内容

hugo new post/blog.md

在centent下创建post/blog.md, blog.md是博客要显示的内容

本地博客

hugo server -t m10c -D
hugo server --theme=m10c --buildDrafts

上面两行命令是执行效果一样的，服务地址

http://localhost:1313/

github远程博客

git创建空仓库

项目名为 你的账户.github.io,如 zhangshan.github.io

打包
hugo -t m10c --baseUrl="https://zhangshan.github.io" -D

执行后会在根目录下生成public文件夹

构建git项目
cd public
git init
git add .
git commit -m "frist commit"
git remote add origin https://github.com/zhangshan/zhangshan.github.io.git
git push -u origin master
访问hugo博客

https://zhangshan.github.io
